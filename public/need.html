Синхронотоп - система учета информации для синхронных турниров по спортивному "Что? Где? Когда?"

Чтобы система стала действительно качественной, надежной, удобной и приятной на вид, нужна помощь с:

  • веб-дизайном
  • тестированием
  • хостингом
Приложение создается на некоммерческой основе - то есть все участники проекта работают безвозмездно. Надеюсь, найдутся люди, которые захотят помочь облегчить труд тех, кто организует турниры по ЧГК.

Тем, кто хочет помочь с дизайном

Напишите мне письмо. Сейчас весь интерфейс исключительно текстовый. Сделать надо, как я понимаю, CSS. Предложения и помощь по улучшению useability тоже принимаются с удовольствием.

Тем, кто хочет помочь с тестированием

Напишите мне письмо. Желательно наличие хотя бы одного из двух качеств:
  • базовые навыки тестирования ПО
  • природная сколонность искать и находить недостатки

    • Тем, кто хочет помочь с хостингом

    Если у Вас есть идеи, как сделать хостинг беслатнным и при этом надежным, чтобы пользование стало бесплатным для всех, напишите мне письмо.
    Приложение написано на Ruby on Rails. Пока что для хостинга я воспользовался сервисom heroku.com. У них есть бесплатный вариант, но когда пользователей будет много, нам его наверняка не хватит. Ориентировочные технические и организационные требования:
    • база данных MySQL на 5-10 MB с ежедневным бэкапом
    • Ruby 1.8.7, Rails 2.3.8
    • для уменьшения нагрузки имеет смысл reverse proxy
    • я даю доступ к коду и конфигурации, в организации хостинга участвую минимально
